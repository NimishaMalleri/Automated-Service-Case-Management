# Automated Service Case Management System

## Overview
This project automates customer complaint handling using Salesforce.

It converts emails into service cases and processes them using OCR and AI concepts for classification, prioritization, and routing.

## Features
- Email-to-Case automation
- OCR-based data handling
- AI-based complaint classification
- Complaint prioritization
- Auto-routing system

## Technologies Used
- Salesforce
- OCR
- Artificial Intelligence
- Automation

## Files Included
- Presentation-1.pptx
- Presentation-2.pptx
- Progress Report.pptx
- Progress Report2.pptx

## Purpose
The project reduces manual effort and improves customer service efficiency.
